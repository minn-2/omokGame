## 파일 구조
omoku_rules.py # 게임 규칙 모듈 (5목 판정, 금수 처리)
omoku_game.py # 게임 핵심 로직 (턴 관리, 승패 판단)
omoku_gui.py # GUI 구현 (사용자 인터페이스)

## 주요 기능

### 기본 기능
- 2인용 오목 플레이 (Human vs Human)
- 바둑판에 클릭으로 돌 배치
- 흑/백 번갈아 가며 진행

### 승리 조건
- 가로 / 세로 / 대각선 **5목 완성 시 승리**

### 규칙 시스템
- 금수 규칙 적용 (장목, 3-3, 4-4 등)
- `omoku_rules.py`에서 규칙 판정 처리

### GUI 기능
- 마우스 클릭 기반 입력
- 현재 턴 표시
- 승리 시 결과 팝업 출력

## 실행 방법
python omoku_gui.py
